# BR Architects

A modern and responsive website built with HTML and CSS.

## 📌 Overview

This project is a responsive landing page inspired by the W3Schools BR Architects template.

It was built to practice:

- Semantic HTML5
- CSS3
- Flexbox
- CSS Grid
- Responsive Design
- Mobile First
- CSS Variables
- Smooth Scrolling
- AOS (Animate On Scroll)

## 🚀 Features

- Responsive layout
- Fixed navigation bar
- Hero section
- Projects gallery
- About section
- Contact form
- Smooth scrolling
- Scroll animations using AOS

## 🛠️ Technologies Used

- HTML5
- CSS3
- AOS Library
